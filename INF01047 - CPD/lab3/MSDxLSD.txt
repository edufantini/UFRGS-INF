Foi utilizada a versão MSD do algoritmo de Radix Sort a fim de preservar a ordem lexicográfica das palavras. Através do algoritmo MSD podemos realizar comparações a partir do primeiro caractere da palavra, utilizando como base de comparação a tabela ASCII. Como as palavras possuem tamanho variável, o algoritmo MSD do Radix Sort é recomendado (controle de fim de sequência), pois podemos maximizar a eficiência do algoritmo não comparando determinadas chaves.
